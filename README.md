Быки и Коровы (авторы: Даниил, Марат, Егор)

Быки и Коровы - логическая игра, в которой нужно угадать секретное число.
Правила игры

    Компьютер загадывает 4-значное число без повторяющихся цифр

    Вы делаете попытки угадать число

    После каждой попытки получаете подсказку:

        Бык - цифра есть в числе и стоит на правильной позиции

        Корова - цифра есть в числе, но стоит не на своей позиции

Пример
text

Загадано: 4271
Ваша попытка: 1234
Результат: 1 бык (цифра 2), 2 коровы (цифры 1 и 4)

Цель игры

Угадать число за минимальное количество попыток!

Удачи в игре!
